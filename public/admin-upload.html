Admin Bulk Upload

Upload a file containing bulk product data. The file should be in one of the following formats: JSON, CSV, or TXT.

Accepted File Formats

  • JSON: A file in JSON format containing product data.
  • CSV: A CSV file with product details in columns.
  • TXT: A text file containing product data in plain text format.

Example File Format

JSON Example:

{
    "products": [
        {
            "name": "Mystery Novel",
            "price": "$19.99",
            "category": "Mystery",
            "image": "images/Mystery.jpg",
            "description": "A thrilling mystery novel."
        },
        {
            "name": "Sci-Fi Adventure",
            "price": "$24.99",
            "category": "Sci-Fi",
            "image": "images/Sci-fi.jpg",
            "description": "An exciting Sci-Fi adventure."
        }
    ]
}

CSV Example:

Name, Price, Category, Image, Description
Mystery Novel, $19.99, Mystery, images/Mystery.jpg, A thrilling mystery novel.
Sci-Fi Adventure, $24.99, Sci-Fi, images/Sci-fi.jpg, An exciting Sci-Fi adventure.

TXT Example:

Mystery Novel|$19.99|Mystery|images/Mystery.jpg|A thrilling mystery novel.
Sci-Fi Adventure|$24.99|Sci-Fi|images/Sci-fi.jpg|An exciting Sci-Fi adventure.
